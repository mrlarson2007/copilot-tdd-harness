# Plugin Packaging Source

This folder contains the distributable Copilot plugin assets used by the
installer and release packaging flow.

- Source projects receive these files under their own `.github/` folder.
- Keep this package tree aligned with the live development copies under the
  repository root `.github/` folder until packaging automation is added.
- Project-specific TDD instructions at
  `.github/instructions/tdd-patterns.instructions.md` are generated per target
  project and are not part of this package source.
